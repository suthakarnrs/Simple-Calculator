# Simple-Calculator
Simple Calculator
A basic web-based calculator designed using HTML, CSS, and JavaScript. This calculator allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division.

Features
Addition (+)
Subtraction (-)
Multiplication (*)
Division (/)
Decimal point (.)
Clear display (C)
Delete last entry (DEL)
Responsive design
Preview

Getting Started
Prerequisites
A web browser to run the HTML file.
Installation
Clone the repository to your local machine.
bash
Copy code
git clone https://github.com/your-username/simple-calculator.git
Open the index.html file in your preferred web browser.
Usage
Click on the calculator buttons to input numbers and perform calculations.
Use the C button to clear the display.
The DEL button deletes the last digit in the display.
Press = to calculate the result.
